Sobre la asignatura

Administración de Bases de Datos I

Sobre la asigntura

Descripción

Los estudiantes han desarrollado sistemas e implementado sus conocimientos de lenguajes de programación en proyectos de cátedra; pero desconocen el proceso para aplicar su software a una solución que gestione los datos almacenándolos en un Gestor de Bases de Datos (DBMS). Se pretende que el estudiante conozca el proceso de modelado e implementación de una base de datos que asegure la confidencialidad, integridad y disponibilidad de los datos.

Competencia

Crear bases de datos consistentes y normalizadas para almacenar la data empresarial aplicando la metodología del diseño, gestión de base de datos, lenguaje SQL y considerando las reglas de negocios desarrolladas de manera individual y colaborativa.

Indicaciones

El ciclo II-2022 se desarrollará en modalidad semipresencial.

Para la asignatura de Administración de Bases de Datos II, se define lo siguiente:

Durante computo I, solo se desarrollan clases teóricas en línea, es decir que no asistirá a prácticas presenciales.

El componente practico de forma presencial se iniciará a partir de computo II y se continuará en computo III, en donde: cada estudiante deberá asistir en el horario que inscribió las prácticas.
